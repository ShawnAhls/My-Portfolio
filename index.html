The Simon Game

The Simon Game is designed as a memory game. The Simon game will play a sequnce of colored buttons that lights up and plays a sound for the player to follow. The game will start when the player clicks on the start button and the game will start with a single color button that will light up and play a sound, the player has to click on that color button. If the player clicks on the correct button the game will add one more color button to the sequence,that the player remember to follow.

My Yummy Dishes

When adding a recipe the user can, if they want to add an image of the recipe they are adding, but if they don't a default image will appear in its place.
